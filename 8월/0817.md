## 느낀점

프론트엔드 부분은 팀원들이 상당부분 개발을 해두어서 내가 하던 부분을 마무리짓고 서버 api 관련 개발을 해보려한다. 프론트엔드에서 담당했던 기능 위주로 api 를 만들어볼 생각인데, 생각보다 이전에 비해 고려할 사항이 많다. 급하지 않게 좋은 코드를 짜보려 노력해야한다.
