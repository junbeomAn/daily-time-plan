## 느낀점

데이터 베이스 모델링은 상당히 어려운 작업인데, 프로젝트의 크기가 더욱 커지니 굉장히 어렵게 느껴진다.
충분한 회의와 요구사항 정의 끝에 이루어져야하는 부분이라고 생각했는데 어쩌면 프론트엔드 개발을 하면서 더욱 정리가 될수도 있겠다는 생각이 든다. 대신 DB를 잘 구성 해둔다면 API 개발에도 도움이 될것이라 생각한다.
